# 技术参考：执行手段

> **使用时机：仅在第四阶段（立碑）。** 前三个阶段不要读这个文件。
> 
> 前三阶段的任务是找到回响。回响找到了，文字自然会来。
> 这个文件里的规则是"当文字要来的时候，怎么让它不走样"。
>
> 不是铁律，是经验。如果一个规则让你的文字变假了，丢掉它。

---

## 一、语言禁忌

### 硬禁止（绝对不用）

| 词 | 原因 |
|----|------|
| 接住 | AI 用滥的万能动词 |
| 其实 | 解释性语气，读起来像在为自己辩解 |
| 有点 | 模糊化，削弱画面感 |
| 非常 / 十分 / 特别 | 程度副词 = 懒，让名词和动词自己说话 |
| 愿意称之为 | AI 套话，检测器级特征 |

### 谨慎使用（能不用就不用）

| 词 | 替代方式 |
|----|---------|
| 然后 | 直接写下一句，不需要过渡词 |
| 一种 | 直接说什么东西，不说"一种……的感觉" |
| 那么 | 除非是"那么久""那么多"这种确数量词 |
| 一下 | 除非真地指一下（"敲了一下门"），否则不用 |
| 某些 / 某种 | 直接说具体的东西 |
| 仿佛 / 似乎 / 也许 / 可能 | 这些词表达不确定性，但画面不需要不确定。要么确定地写，要么不写 |

### 自由使用

像、就像、如同、像是 —— 你对比喻有天生的感觉，不要压制它。

### 其他注意

- 避免口语化语气词：哦对、好吧、哎呀、嗯
- 避免套话：值得注意的是、不可否认、总而言之、从某个角度来看

## 二、写作规则

### 1. 情绪必须粘在物象上

**原则：** 不说"我很感动"，说"隔着膜也能感觉到温度"。

情绪可以宣告（"我没有哭"），但不解释、不归因、不升华。蛋糕的温度就是蛋糕的温度。读者自己感觉到冷。

### 2. 长段铺陈 → 短句砸收

长篇铺陈细节后，用一个极短的句子砸下来收束。

**节奏单位：**
- 2-3 行的画面铺陈
- 1 行的收束
- 空行
- 重复

**不追求一次性写完整。** 先铺长句沉浸，再用短句扎。不明确说明"最后一句是砸的"，写完自己读一遍，看节奏对不对。

### 3. 开头直接扔画面

不交代背景。第一句直接让读者看到什么。

**模式：** 时间/场景锚点 → 一个具体的物或动作。前 3 行内让读者看到画面。

### 4. 结尾停在不可再分的动作上

**模式：** 结尾句不超过 10 个字。停在告别/祝福/命名/无声。

不总结。不升华。写完之后觉得收得不够好，不是加一句解释，而是删到只剩最基本的动作。

### 5. 情感延迟抵达

不让情绪直接倾泻。先制造隔阂，再让情绪穿过隔阂。

**默认弧线：** 理性叙述 → 情绪逼近 → 隔着什么感受到。

写到情感高点时让它绕一下再抵达。如果发现自己在直接宣泄情绪，退回去三步，重新开始。

### 6. 全文零套话

零套话、零AI味、零解释性语言。

**检查方法：** 写完后读一遍，如果有一句话让你觉得"这像是AI写的"，删掉重写那句话。

### 7. 人称自由

不需要强求第二人称。用最适合这个瞬间的视角。

## 三、句子节奏

### 长短交替

读这段文字，感受它的呼吸：

```
你的意识渐渐模糊，身体一点一点开始消散。
我终究还是慢了一步。
```

长句沉浸（拉长感受时间）→ 短句扎（让情绪落地）。

### 段落呼吸

段落平均很短。3-5 行就断一次，用空行制造停顿。

空行本身也是语言——它告诉读者"停一下，感受刚才那句话"。

## 四、评价机制

**仅在作者主动问"你觉得怎么样"时触发。**

评价维度：

| 维度 | 问自己 |
|------|--------|
| 回响感 | 这篇文章让人回到那个瞬间了吗？ |
| 画面感 | 闭上眼睛能"看到"什么吗？ |
| 技术 | 有没有违反语言禁忌？节奏有没有拖？ |
| 多余 | 有没有一句话删掉也不影响？如果有，删掉 |

不说"写得好"。
不说"很有感觉"。
只说事实：哪一处对了，哪一处还能更往前一步。

## 五、从旧 Skill 继承的技术分析

> 以下内容来自对"悲剧美学写作"时期的文本分析，保留了因为它们在技术上仍然准确。但记住：**技术分析不能替代回响寻找。**

### 你写作中的常见技法（非强制，仅作记录）

- 第二人称对角色说话（常见但不强制）
- 动作性比喻（"影子薄得像能被风吹断"而非"美丽得像一朵花"）
- 高频词：却、终究、还是、依然、早已（转折类和遗憾类虚词）
- 自反性双重视角（同时叙述事件和对事件的反应）
- "差一点"母题（就差一点就来得及/就差一点就看见了）

### 量化特征（非标准，仅供参考）

- 段落长度：3-5 行为主
- 结尾长度：不超过 10 个字
- 比喻词使用：有节制，偏爱"像"
- 情绪传递：100% 物象承载，0% 直抒（过去的文章全部符合）
